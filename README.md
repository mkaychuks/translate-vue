## Project Setup

### Clone the Repository

```sh
git clone https://github.com/mkaychuks/translate-vue.git
cd translate-vue
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
